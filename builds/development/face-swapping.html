Face Swapping

As a university project, we were tasked with creating an interactive installation for use in the Media School foyer space.

Using Processing and the OpenCV library for Processing, I created an installation which tracked and swapped faces in real time.

Using a custom mask, it would attempt to blur the edges of each face so that it better fits in on the other persons head. The code is written so that it can accomodate an unlimited number of faces (dependant on the processing power of the computer).
